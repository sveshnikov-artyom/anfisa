reccnt¶

Aspect-based full view presentation of variant

Synopsis¶

reccnt

Arguments:

ds: dataset name

rec: record order number

details: optional 0-1 bitmap representing selected transcripts

Return value: View Record structure

Description¶

The request prepares view presentation of variant record by its order number in dataset.

In context of workspace filtering transcripts rendering logic can highlight selected transcripts if details argument is set. Value of details argument the client receives in ws_list response in structure Record descriptor.
