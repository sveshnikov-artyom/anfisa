export¶

Export operation

Synopsis¶

export

Arguments:

ds: dataset name

filter: optional name of applying filter

conditions: optional list of applying
condition descriptors in JSON string representation
zone: optional zone descriptor
in JSON string representation

Return value:

{ dictionary
“kind”: "excel"
“fname”: URL to excel file to download
}

Description¶

The request creates exported Excel file for selected variants. Selection should be of limited size: not more than 300 variants.

Selection is defined by:

  • filter applies if either filter or conditions is set (see discussion
    here)
  • zone applies if zone is set - actual for workspaces only.
    (Nonempty zone list can contain special pair of strings ["NOT", true]. In this case zone option works in reverse context: records satisfying condition on zones are filtered out.)

Resulting file is created on servers side and is ready for download from moment of creation and later.

(Cleaning of these files are subject of Anfisa administration)
