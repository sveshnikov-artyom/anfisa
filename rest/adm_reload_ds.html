adm_reload_ds¶

Force dataset reload

Synopsis¶

adm_update

Arguments:

ds: dataset name

Return value: "Reloaded <dataset name>" as plain string

Description¶

The method reloads dataset in memory.

Comment¶

The method can be used in administrative work, especially if dataset set of solutions is changed by app.adm_mongo - MongoDB storage administration utility.
