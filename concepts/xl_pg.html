XL-dataset work page¶

XL-dataset page is essentially simpler than Workspace page, and contains only subset of ws-functionality. But in context of XL-dataset evaluations might be heavy, so the main focus of the Front End application support for this page is an issue of heavy evaluations and long delays.

The page supports the following functionality issues:

  • Full complex of Filtering regime with its features:

    • Status report mechanism with delays
    • support filters as Solution items in work
    • the logic of regime requires top detailization of information, so transcripts are used as filtration items instead of variants

  • Auxiliary viewing regime as the main regime

  • operation support:

    • creation of secondary workspace
    • Export operation: form and provide access to download Excel document with properties of filtered variants, in case if count of variants is limited (up to 300 variants).

REST API requests¶

For support WS-dataset work page:

  • dsinfo
    Dataset information
  • ds_list
    List of variants in in auxiliary viewing regime
  • reccnt
    Aspect-based full view presentation of variant
  • export
    Export operation
  • ds2ws
    Creation of secondary workspace
  • see also Filtering regime
