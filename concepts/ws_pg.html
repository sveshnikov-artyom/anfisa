Workspace page¶

WS-dataset page is the most complex of working pages, because it is used by the user for the most important goals: to detect single variant (or small number of variants) as a final result of dataset analysis, from large (but comparatively small, up to some thousands of variants) dataset.

The page supports the following functionality issues:

  • Full viewing regime as the main regime

  • Full complex of Filtering regime with its features:

    • Status report mechanism with delays
    • support filters as Solution items in work
    • the logic of regime requires top detailization of information, so
      transcripts are used as filtration items instead of variants
  • manual user work with tagging tools to associate variants

    with tags and notes

    Front End can support tools to navigate the user between variants with fixed tag. Take care with these tools: the user can drop tag from variant and might be confused by tag navigation response.

  • Zone panel that provides “secondary” filtration tool that can be

    combined with Filtering regime

  • operation support:

    • creation of secondary workspace
    • Export operation: form and provide access to download Excel document
      with properties of filtered variants, in case if count of variants is limited (up to 300 variants).
  • Special feature of Workspace page: Rules are available in Filtering regime.

    Rules is special aggregated multiset property that detects what decision trees are positive on the variant.

    This issue closes up long circle of functionality logic: decision trees are complex solution items that can be used for detection for variety of properties of variants, so at the WS-dataset working page the user can apply all of them to get final result.

REST API requests¶

For support WS-dataset work page:

  • dsinfo
    Dataset information
  • ws_list
    Current list of variants
  • reccnt
    Aspect-based full view presentation of variant
  • zone_list
    Zone support information
  • ws_tags
    Tagging variant information retrieval and modifications
  • export
    Export operation
  • ds2ws
    Creation of secondary workspace
  • see also Filtering regime
